# Uizard | AI-Powered UI Design Tool | Key Features | Pricing | How the Architect Agent Uses it

## What is it?

Uizard is an AI-powered UI design tool that allows users to generate UI designs from text prompts or screenshots. It also offers features for rapid prototyping, Figma plugin integration, screenshot-to-wireframe conversion, sketch-to-mockup conversion, and AI text writing.

## Key Features

*   AI-powered UI design generation from text prompts or screenshots.
*   Rapid prototyping: creating clickable prototypes in seconds.
*   Figma plugin: facilitating UI/UX design process.
*   Screenshot-to-wireframe conversion.
*   Sketch-to-mockup conversion.
*   AI text writer.

## How the Architect Agent Uses it:

The Architect agent can leverage Uizard to rapidly generate UI mockups and prototypes based on user requirements or specifications. This can speed up the design process and allow for quick iteration on different design concepts. The Architect agent can also use the Figma plugin to integrate Uizard generated designs into existing Figma projects.
